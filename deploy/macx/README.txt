================================================================================
CasparCG Client 2.0b2
================================================================================
Thank you for your interest in CasparCG Client. The included software
is provided as-is by Sveriges Televison AB.

Version 2.0b2 is released for Windows and OS X at this time. Later beta versions will 
also be released for Linux.

More information at http://casparcg.com/




SYSTEM REQUIREMENTS
===================
The Windows version should work with Windows XP or newer.
The Mac version should work with OS X 10.5 and newer.


INSTALLATION WINDOWS
====================
No installation required, just start the 'CasparCG Client 2.0b2' shortcut.


INSTALLATION OS X
=================
Open the DMG file and drag the 'CasparCG Client 2.0b2' app to your applications folder.


CHANGES IN 2.0b2
================
- First Mac version.
- Fixed various bugs.
- Various GUI tweaks.


CHANGES IN 2.0b1
================
- Fixed various bugs.
- Added support for open/save rundown lists.
- Added support for file recording using qtrle, dnxhd, prores, ffvhuff rawvideo, h264.
- Added support for Image scroll producer.
- Added support for DeckLink input.
- Added support for transactional mixer commands using commit/defer.
- Added support for GPI/GPO on groups and individual rundown items.
- Added support for GPI/GPO. Two sample project can be found in the 
  CasparCG SVN repository (arduino/iocore).
